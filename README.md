# Usage

##### Clone the repo
```bash
git clone https://github.com/Ahish9009/ash/
```
##### Navigate into it 
```bash
cd ash
```
##### Generate the binary executable
###### On Linux:
```bash
make linux
```
###### On MacOS:
```bash
make mac
```
##### Execute 
```bash
./ash
```
